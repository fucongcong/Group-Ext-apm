# ext_group-apm
a php extension for monitor your code performance

#### Install

    phpize
    ./configure
    make && make install

#### Get start

```php 
    <?php
    /*
     * your code
     * do something
     *
     *
     */

    //get the monitor
    $data = group_apm();

```

#### example1

    php example/group_res.php

#### example2

    php example/group_apm.php
